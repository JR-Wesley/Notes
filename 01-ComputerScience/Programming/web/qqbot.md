# Guide

在<a href=" https://q.qq.com/#/app/bot">QQ 开放平台</a>创建机器人，添加到测试群。获得机器人 ID，可定制基本信息。添加指令信息，在 QQ 端的机器人个人界面显示。

在机器上配置环境，配置中修改一些网站 API、添加创建的机器人 ID。
